所有配置参数，分为两个3部分，1帧同步 2日志 3指令行.(其中1和2均可在main.go文件中修改)
1.帧同步配置请点击
2.日志
日志目录必须是777
日志目录下必须还得有4个文件夹：app backend frontend service，同样需要777
OutFileFileName文件名
Level级别，如：DEBUG ALL
OutTarget屏幕 文件 网络
OutContentTypenormal json
OutFileHashType文件存储方式，如：年 月 日 小时
具体参数有点多，可参考 ： zlib.log
3.指令行
启动指令：go run main.go local 127.0.0.1 2222 2223 2224 /data/www/golang/src/logs
